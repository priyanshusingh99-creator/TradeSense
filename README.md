# TradeSense

TradeSense is a full-stack stock trading platform inspired by Zerodha. It includes a landing website, trading dashboard, holdings and positions views, and backend APIs for portfolio data and order placement.

## Features

- Landing website with Home, About, Products, Pricing, and Support pages
- Trading dashboard UI
- Holdings and positions pages
- Buy order action support
- Backend API built with Express.js
- MongoDB support
- Sample data fallback for local development
- One-command local startup

## Tech Stack

- React.js
- React Router
- Material UI
- Chart.js
- Node.js
- Express.js
- MongoDB
- Mongoose
